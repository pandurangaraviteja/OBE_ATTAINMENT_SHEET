📐 NAAC / NBA Attainment Formula Reference ▼

Step 1 — Question Normalisation

Nij = Mij / MMj

Nij = normalised score, student i, question j
Mij = marks obtained (absent = 0)
MMj = max marks for question j

Step 2 — Student CO Score (per exam type)

COi,k,t = (1/n) × Σ Nij

n = questions mapped to COk in exam type t.
If no questions exist for a CO in type t, that type is excluded — its weight is redistributed.

Step 3 — Weighted CO Score (combined)

COi,k = Σt (wt / Σwavail) × COi,k,t

Weights redistributed among exam types that actually have questions for COk. Prevents artificial deflation from missing exam types.

Step 4 — Class CO Attainment (Direct)

COclass,k = (1/S) × Σi=1..S COi,k

S = total students. This is the direct attainment value reported to NAAC/NBA.

Step 5 — CO Attainment Level (COAL)

Level 3: COclass ≥ 70%
Level 2: COclass ≥ 60%
Level 1: COclass ≥ 50%
Level 0: COclass < 50%

Thresholds configurable. NAAC recommends 60–70% as the target band.

Step 6 — PO Attainment (Indirect)

POp = Σk(COclass,k × Wkp) / ΣkWkp

Wkp ∈ {1,2,3} = CO–PO mapping strength. Only COs with Wkp > 0 contribute. NBA weighted average method.

Bloom's Taxonomy (CDLL)

L1 Remember · L2 Understand
L3 Apply · L4 Analyse
L5 Evaluate · L6 Create

Each CO is mapped to a Bloom's level. CDLL shows the cognitive depth profile — higher L4–L6 = HOTs-aligned curriculum per NAAC criterion.

CDAL — Cognitive Attainment by Bloom Level

CDALb = avg(COclass,k) ∀k : Bloom(k)=b

Groups COs by Bloom level; computes class-avg attainment per cognitive tier. Used in NAAC SSR to show HOTs vs LOTs achievement gap.
